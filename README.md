# Trafik Takip Sistemi

iPhone ve bilgisayarda çalışan PWA.

## V2 yenilikleri
- Osmangazi, Nilüfer ve Yıldırım mahalle listeleri
- İlçeye göre otomatik mahalle seçimi
- Ekip eklerken ilçe ve birden fazla görev mahallesi seçimi
- Ekibin görev mahallelerini ekip kartında gösterme
- Kayıt girerken ekibin atanmış mahallelerini yıldızlı olarak en üstte gösterme

## Güncelleme
GitHub repository içindeki eski dosyaların yerine bu ZIP içindeki dosyaları yükleyin.
Pages yayını birkaç dakika içinde otomatik güncellenir. iPhone Safari eski sürümü gösterirse
sayfayı yenileyin veya uygulamayı tamamen kapatıp tekrar açın.

Veriler cihazın tarayıcı hafızasında tutulur. Safari verileri silinirse kayıtlar da silinir.
