# Trafik Takip Sistemi

iPhone ve bilgisayarda çalışan basit PWA.

## Yayınlama
1. GitHub'da `trafik-takip` adlı public repository oluşturun.
2. Bu klasördeki tüm dosyaları repository ana dizinine yükleyin.
3. Settings > Pages bölümünde Source olarak `Deploy from a branch` seçin.
4. Branch: `main`, klasör: `/root`, ardından Save.
5. Oluşan adresi iPhone Safari'de açın.
6. Paylaş > Ana Ekrana Ekle.

Veriler cihazın tarayıcı hafızasında tutulur. Safari verileri silinirse kayıtlar da silinir.
